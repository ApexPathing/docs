# Dist - Class
---

The `Dist` class is used to represent distances in Apex Pathing. It provides a 
clear and consistent way to provide for units and methods. This class makes use
of the `DistUnit` enum to specify the unit of measurement for distances. You can
read more about the `DistUnit` enum in the [DistUnit - Enum](../dist-unit-enum)
documentation. Distance values are stored internally as a double in inches.

See the class on [GitHub](https://github.com/ApexPathing/ApexPathing/blob/main/TeamCode/src/main/java/geometry/Dist.java)
for more information about the implementation of the `Dist` class.

## Construction
- `Dist.of(double value, DistUnit unit)`: Creates a new `Dist` object with the 
specified value and unit of measurement.
- `Dist.fromIn(double value)`: Creates a new `Dist` object from a value in inches.
- `Dist.fromFt(double value)`: Creates a new `Dist` object from a value in feet.
- `Dist.fromM(double value)`: Creates a new `Dist` object from a value in meters.
- `Dist.fromCm(double value)`: Creates a new `Dist` object from a value in centimeters.
- `Dist.fromMm(double value)`: Creates a new `Dist` object from a value in millimeters.
- `Dist.zero()`: Creates a new `Dist` object representing a distance of zero.

## Getters
- `get(DistUnit unit)`: Returns the distance value in the specified unit of measurement.
- `getIn()`: Returns the distance value in inches.
- `getFt()`: Returns the distance value in feet.
- `getM()`: Returns the distance value in meters.
- `getCm()`: Returns the distance value in centimeters.
- `getMm()`: Returns the distance value in millimeters.

## Operations
- `plus(Dist other)`: Returns a new `Dist` object representing the sum of this distance and the specified distance.
- `minus(Dist other)`: Returns a new `Dist` object representing the difference between this distance and the specified distance.
- `times(double factor)`: Returns a new `Dist` object representing this distance multiplied by the specified factor.
- `times(Dist other)`: Returns a new `Dist` object representing this distance multiplied by the specified distance.
- `div(double divisor)`: Returns a new `Dist` object representing this distance divided by the specified divisor.
- `div(Dist other)`: Returns a new `Dist` object representing this distance divided by the specified distance.
- `abs()`: Returns a new `Dist` object representing the absolute value of this distance.
- `hypot(Dist other)`: Returns a new `Dist` object representing the hypotenuse of this distance and the specified distance.
- `mirror()`: Returns a new `Dist` object representing the mirrored value of this distance (i.e., multiplied by -1).
- `copy()`: Returns a new `Dist` object that is a copy of this distance.
